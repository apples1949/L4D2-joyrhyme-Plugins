# Melee_Auto_Thirdperson 近战自动第三人称
#### 切换近战时自动第三人称. 且会记录个人的开关状态(例如关闭后,下次进服也是默认关闭)
---
* 所需平台
    * Sourcemod 1.11+
---
* 插件指令
    * melee_auto_3th_enable - **近战自动第三人称总开关**
    * melee_auto_3th_chainsaw - **切换到电锯是否自动第三人称**
---
* 插件命令
    * /auto_third -- **开关自身的近战第三人称**
